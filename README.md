# Quran Galaxy

Quran Galaxy adalah aplikasi web interaktif untuk menjelajahi topik-topik Al-Qur’an dalam bentuk galaksi 3D. Setiap topik divisualisasikan sebagai node bercahaya, terhubung dengan topik lain, dan dapat dibuka untuk membaca ayat-ayat terkait.

## Fitur

- Galaksi 3D topik Al-Qur’an
- Pencarian topik dalam Bahasa Indonesia, Inggris, dan Arab
- Panel ayat dengan teks Arab, terjemahan Indonesia, tafsir singkat, dan audio murottal
- Mode Explore, Quiz, Pencapaian, dan Mind Map 2D
- Penyimpanan progres memakai localStorage
- Desain responsive untuk desktop dan mobile

## Teknologi

- HTML, CSS, Vanilla JavaScript
- Three.js
- OrbitControls
- Tailwind CSS CDN
- Google Fonts: Scheherazade New, Inter, Fraunces

## Cara Menjalankan

Cukup buka file `index.html` di browser modern seperti Chrome, Firefox, Safari, atau Edge.

Tidak perlu backend, database, framework, atau build step.

## Data

Aplikasi mengambil data Al-Qur’an dari sumber JSON publik dan menyimpannya di cache lokal browser agar pemuatan berikutnya lebih cepat.

## Catatan

Quran Galaxy adalah alat eksplorasi dan tadabbur tematik. Pemetaan topik bersifat kuratorial dan belum dimaksudkan sebagai rujukan tafsir final atau fatwa.

## Lisensi

Silakan gunakan, pelajari, dan kembangkan dengan tetap menjaga adab terhadap teks Al-Qur’an.
